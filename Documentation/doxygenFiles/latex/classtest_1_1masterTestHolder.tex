\subsection{Detailed Description}
Unit Test singleton. 

This class contains a set of library tests. Every library test must add itself to this class in-\/order to be tested. The \doxyref{test\+::master\+Test\+Holder\+::run\+Tests()}{p.}{classtest_1_1masterTestHolder_aaf3fe2e7899371abf2ad03505e8fcc7d} function runs all of the library tests.
